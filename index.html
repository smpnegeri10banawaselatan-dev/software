Apa itu Software?

Software adalah kumpulan instruksi atau program yang digunakan untuk mengoperasikan komputer dan menjalankan tugas-tugas spesifik.

Contoh Software Aplikasi:

  • Microsoft Word
  • Adobe Photoshop
  • Google Chrome

Contoh Software Sistem:

  • Windows 10
  • Linux Ubuntu
  • macOS

Fungsi utama software sistem adalah?

  • Menggambar
  • Menjalankan komputer dan perangkat keras
  • Menonton video

Sistem operasi termasuk dalam jenis software?

  • Aplikasi
  • Game
  • Sistem

Contoh Software Sistem:

  • Windows 10
  • Linux Ubuntu
  • macOS
